797 #### Basic Stepsize Control
798

804 ```

809 `reltol` scales with the size of the dependent variables and so one can interpret
810 `reltol=1e-3` as roughly being (locally) correct to 3 digits. Note tolerances can
811 be specified element-wise by passing a vector whose size matches `u0`.
